# Back-end Cafétéria

Ce dossier contient l'API REST pour la gestion de la cafétéria.

## Stack technique
- Node.js + Express.js
- Authentification JWT
- Hash des mots de passe avec bcrypt
- Base de données : à déterminer (MongoDB recommandé, adaptable SQL)

## Structure du projet

```
src/
  controllers/
  models/
  routes/
  middlewares/
  utils/
  app.js
.env
package.json
README.md
```

## Fonctionnalités principales
- Authentification (inscription, connexion)
- Gestion des utilisateurs
- Gestion des achats
- Gestion du stock
- Rôles (admin/utilisateur)
